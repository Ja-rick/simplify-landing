Privacy Policy

Simplify 2.0 does not collect or store any personally identifiable information. The extension only sends selected text to an external API in order to simplify it.

This data is not stored or shared with any third parties. API usage is monitored solely for billing and rate-limiting purposes. No user identity, activity, or browsing history is recorded.
